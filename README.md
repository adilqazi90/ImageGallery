# 🖼️ Image Gallery Web Application

## 📌 Project Overview

This project was developed as part of the **Frontend Development Internship at CodeAlpha**.

The Image Gallery is a modern and responsive web application built using **HTML, CSS, and JavaScript**. It allows users to browse images in an organized gallery layout with smooth interactions and an intuitive user experience.

---

## 🚀 Features

- Responsive image gallery layout
- Modern and attractive user interface
- Hover effects and smooth animations
- Image preview functionality
- Mobile-friendly design
- Clean and organized code structure
- Fast and lightweight performance

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
